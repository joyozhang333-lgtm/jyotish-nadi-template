# Nadi 资料来源

## 结论先说

能拿到一部分，但不是“公开完整、可直接训练的个人叶片库”。

当前能拿到的主要是三类资料：

1. 官方机构的棕榈叶手稿目录与部分扫描件
2. 与 `Jyotisha / Astrology` 相关的手稿库和目录
3. 可替代的 `Nadi` 系统出版物、译本和研究线索

当前没有找到的，是：

- 公开、可信、可下载的 `thumb impression -> bundle -> leaf` 对照数据集
- 完整公开的 `Agastya / Bhrigu / Suka` 个体叶片库
- 适合直接拿来训练“真实找叶”的标准标注集

## 1. 官方棕榈叶与手稿库

### Government Oriental Manuscripts Library, Chennai

- 官方手稿入口：
  - [GOML Manuscript](https://www.tnarch.gov.in/goml/manuscript)
  - [Online Manuscripts](https://www.tnarch.gov.in/online-scripts)
- 价值：
  - 官方馆藏
  - 有 palm-leaf / paper manuscript 元数据
  - 有部分在线扫描 PDF
- 当前意义：
  - 适合做目录级检索
  - 适合找 `Astrology / Jyotisha` 相关条目
  - 不等于已拿到“个人纳迪叶片匹配库”

### 一个官方在线样例

- [GOML 手稿详情页示例](https://www.tnarch.gov.in/goml/manuscripts/view/267)
- 这个页面直接展示了：
  - `Type of Manuscript = Leaf`
  - `Subject = Astrology`
  - 可跳转 `View PDF`

## 2. 更大范围的手稿资源中心

### National Mission for Manuscripts

- [Manuscript Resource Centres](https://www.namami.gov.in/manuscript-resource-centres)
- [Manuscript Database / Pandulipi Patala](https://namami.gov.in/main-category/manuscripts)
- 价值：
  - 能定位大型手稿馆藏
  - 能看到哪些馆藏覆盖 `jyotisha`
  - 能帮助我们建立“资料来源地图”

从官方资源页可确认的相关馆藏包括：

- `Thanjavur Maharaja Serfoji's Saraswati Mahal Library`
- `Oriental Research Institute and Manuscripts Library, University of Kerala`
- 这些馆藏都明确覆盖 palm-leaf manuscripts，并包含 `jyotisha` 类手稿。

## 3. 可直接利用的等价资料

### Tamil Digital Library 的目录级线索

- [Catalogue of Palm Leaves](https://www.tamildigitallibrary.in/admin/assets/book/TVA_BOK_0021115_Palm_Manuscripts.pdf)
- 这个目录里能直接看到类似条目：
  - `Varagarishi Nadi Mattrum`
  - `Vaalmiki Bhooda Nadi`
  - `Vaalmiki Naadi`
- 价值：
  - 可以拿来做“有哪些 Nadi 名称实际出现在目录里”的证据层
  - 适合建设术语表、文本来源表、主题映射表

### Dhruva Nadi 的次级资料线索

- [L. D. Swamikannu Pillai PDF](https://www.tamildigitallibrary.in/admin/assets/book/TVA_BOK_0024173/TVA_BOK_002417_L_D_Swamikannu_Pillai.pdf)
- 这个 PDF 的出版物列表里能看到 `Maximum Age of Dhruva Nadi, 1923`
- 价值：
  - 说明 `Dhruva Nadi` 至少有明确的现代研究或讨论线索
  - 适合做文献线索追踪

### Bhrigu Nandi Nadi 的出版物

- [Open Library: Bhrigu Nandi Nadi](https://openlibrary.org/books/OL11495304M/Bhrigu_Nandi_Nadi)
- [Google Books: Bhrigu Nandi Nadi](https://books.google.com/books/about/Bhrigu_Nandi_Nadi.html?id=Xq9huAAACAAJ)
- 价值：
  - 适合做“等价资料层”
  - 可以支持我们先做规则层、术语层、案例层，而不是直接宣称真实叶片匹配

## 4. 对产品最有用的资料分层

### A 层：官方目录 / 扫描件

适合做：

- 术语表
- 主题表
- 手稿来源映射
- 文献证据页面

### B 层：公开出版物 / 译本 / 研究资料

适合做：

- Nadi 规则层
- 章节模板
- 专题包补强
- 解释风格校正

### C 层：真实叶片匹配所需但当前缺失的数据

缺的是：

- thumb 分类体系
- bundle 索引
- leaf-level 标注
- OCR / 转写 / 翻译对齐
- 人工复核流程

## 当前最现实的产品策略

1. 用官方目录和手稿馆藏做 `证据来源层`
2. 用公开 Nadi 出版物做 `等价资料层`
3. 继续保持产品表述为：
   - `Nadi 风格专业 skill`
   - 不是 `真实叶片匹配系统`
